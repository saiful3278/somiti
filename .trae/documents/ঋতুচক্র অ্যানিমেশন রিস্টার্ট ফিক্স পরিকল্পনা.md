## সমস্যা ও লক্ষ্য
- সমস্যা: প্রতিটি FX অ্যানিমেশন (বৃষ্টি/তুষার/পাতা/পাপড়ি) মাঝপথে পুনরায় চালু হচ্ছে; পুরোপুরি নিচে পৌঁছে উপরের থেকে নতুন রান শুরু হচ্ছে না।
- লক্ষ্য: প্রতিটি অ্যানিমেশন উপর থেকে শুরু → পুরোপুরি নিচে পৌঁছানো → অফ-স্ক্রিনে সংক্ষিপ্ত বিরতি → উপরের থেকে পুনরায় শুরু; একবার স্ক্রল-এন্ট্রি হলে অ্যানিমেশন mid-frame এ pause/resume হবে না।

## মূল কারণ
- IntersectionObserver দিয়ে `.in-view` টগল করার সময় `animation-play-state` pause/resume হওয়ায় frame মাঝখানে থেমে আবার resume হচ্ছে।
- `repeating-linear-gradient` ভিত্তিক `fx-rain`–এ ছোট লুপের কারণে নিচে পৌঁছানোর ফিল আসে না।

## সমাধান সারাংশ
1. JS-এ একবার মাত্র স্টার্ট: `.in-view` টগল না করে `.animated` ক্লাস একবার যোগ করা হবে; observer `unobserve` করবে।
2. CSS keyframes multi-phase: প্রতিটি `fx-*`-এর জন্য 0–90% downfall, 90–92% off-screen dwell, 92–100% reset-top; `animation-iteration-count: infinite`, `animation-direction: normal`।
3. Rain implementation বদল: gradient-shift বাদ দিয়ে pseudo-particles (box-shadow ডুপ্লিকেট) দিয়ে vertical drop; বড় travel distance যাতে পুরো কার্ডের উচ্চতা কভার হয়।
4. Randomization: কার্ডভিত্তিক `--fx-delay`/`--fx-speed` ভ্যারিয়েবল দিয়ে delay/duration vary করে desync, যাতে একসাথে restart দেখায় না।
5. Performance: `will-change: transform`, `contain: paint` বজায় রাখা; `prefers-reduced-motion` হলে FX নিষ্ক্রিয়।

## কোড পরিবর্তন
### 1) `src/pages/LandingPage.jsx`
- Observer লজিক (প্রায় 24–46 লাইন):
  - `.season-item` এ `.animated` ক্লাস যোগ করা মাত্র; `.in-view` টগল বাদ।
  - `threshold: 0.6`, `rootMargin: '0px'`; প্রতিটি item `io.unobserve(el)` করা হবে add হওয়ার পর; console log রাখা হবে (`[LandingPage] season animated-start`).
- ঋতুচক্র কার্ড (প্রায় 136–146 লাইন):
  - প্রতি `.season-item`-এ style ভ্যারিয়েবল সেট: `style={{'--fx-delay': '0.6s', '--fx-speed': '7s'}}` ইত্যাদি; hover লগ অপরিবর্তিত।

### 2) `src/styles/LandingPage.css`
- `.season-item .fx`:
  - `animation-play-state: running` ডিফল্ট; `.animated .fx`-এ animation অন থাকবে; `.in-view` নির্ভরতা সরানো।
- Keyframes (নতুন ও বিস্তৃত):
  - `@keyframes rainDrop`: 0% top → 90% bottom; 92%–100% reset-top; linear।
  - `@keyframes snowFlake`: একই ধাঁচে; বিভিন্ন সাইজের কণার জন্য আলাদা delay।
  - `@keyframes leafDrift`: Y-translation + X-oscillation + rotate; 0–90% drift down; dwell; reset।
  - `@keyframes petalRise`: উল্টো দিকে (বসন্ত) — bottom থেকে top; একইভাবে dwell ও reset।
- `fx-rain`/`fx-snow`/`fx-leaves`/`fx-blossom` ক্লাস:
  - pseudo elements (`::before`, `::after`) + `box-shadow` clusters; `animation: rainDrop var(--fx-speed) linear infinite; animation-delay: var(--fx-delay)`।
- `prefers-reduced-motion`: সব `.fx` অ্যানিমেশন `none`।

## উদাহরণ CSS স্নিপেট (ধারণা)
- Rain:
```
.fx-rain::before { content:""; position:absolute; left:50%; top:-10%; width:2px; height:8px; background:rgba(59,130,246,0.5); box-shadow: -80px -40px rgba(59,130,246,0.5), 0 -120px rgba(59,130,246,0.4), 80px -60px rgba(59,130,246,0.45);
  animation: rainDrop var(--fx-speed, 2.4s) linear infinite; animation-delay: var(--fx-delay, 0s);
}
@keyframes rainDrop { 0% { transform: translateY(0); } 90% { transform: translateY(180%); } 92% { transform: translateY(200%); opacity: 0; } 100% { transform: translateY(0); opacity: 1; } }
```
- Snow:
```
.fx-snow::before { /* multiple flakes via box-shadow */ animation: snowFlake var(--fx-speed, 6s) linear infinite; animation-delay: var(--fx-delay, .3s); }
@keyframes snowFlake { 0% { transform: translateY(0); } 90% { transform: translateY(200%); } 92% { transform: translateY(240%); opacity:.0; } 100% { transform: translateY(0); opacity:1; } }
```
- Leaves:
```
@keyframes leafDrift { 0% { transform: translateY(0) translateX(0) rotate(0); }
  50% { transform: translateY(100%) translateX(-6%) rotate(12deg); }
  90% { transform: translateY(200%) translateX(-12%) rotate(24deg); }
  92% { opacity:0; }
  100% { transform: translateY(0) translateX(0) rotate(0); opacity:1; }
}
```

## গ্রহণযোগ্যতার মানদণ্ড
- কোনো FX mid-run-এ পুনরায় চালু/রিসেট হবে না; সবগুলো নিচে পৌঁছে তারপর উপরের থেকে শুরু হবে।
- স্ক্রল-ইন হওয়ার পর অ্যানিমেশন একবার শুরু হবে; `.animated` ক্লাস থেকেই চালু থাকবে; বাইরে গেলে pause হবে না।
- `prefers-reduced-motion` সক্রিয় থাকলে FX বন্ধ।
- Console logs: hover, animated-start, observer ডিসকানেক্ট — সব লগ থাকবে।

## পারফরম্যান্স ও UX
- GPU-friendly transform-only keyframes; কম DOM; `contain: paint`।
- Delay/speed ভ্যারিয়েবল দিয়ে desync; একসাথে রিস্টার্টের ভিজ্যুয়াল কমে যাবে।
